# Revolution2

Unreal Engine C++ 项目，包含多种玩法变体与联机会话插件。

- 入门指南：`docs/getting-started.md`

- 模块文档：`docs/variant-shooter-weapons.md`

目录要点：
- `Source/` 游戏模块源码
- `Content/` 关卡与资源（`Maps/`, `Variant_Shooter/`, `Variant_Horror/`, `FirstPerson/`）
- `Plugins/MultiplayerSessions/` 联机会话插件
- `Config/` 项目配置

构建、运行、打包与常见问题详见入门指南。
